A simple chess game built using React on codesandbox.io.

I built this game because I thought it would be an interesting challenge and to improve my React ability. If I would have to remake this game, I would try to give each function a single purpose. I found myself changing or re writing entire sections of code because most of the functions I wrote were too specialized for one task.

The current game is very simple, the opponent will only play random moves and doesn’t take into account the current position when making a move. I will be expanding the number of different opponents the player can challenge each with a different playstyle and difficulty.
